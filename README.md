# 📚 BiblioTEK

Application web de gestion d'une bibliothèque développée avec **Laravel** dans le cadre du **BTS SIO SLAM**.

Le projet permet de gérer les livres, les exemplaires, les usagers et les emprunts au sein d'une bibliothèque.

---

# 🎯 Objectif du projet

L’objectif de l’application est de permettre :

- la gestion du catalogue de livres
- la gestion des exemplaires disponibles
- l’emprunt et le retour de livres
- la gestion des usagers
- l’administration via un back-office sécurisé

Le projet respecte une architecture **MVC (Model - View - Controller)** et utilise une **base de données relationnelle MySQL**.

---

# ⚙️ Technologies utilisées

- **PHP 8**
- **Laravel 12**
- **MySQL**
- **Blade (templating Laravel)**
- **HTML / CSS**
- **Git / GitHub**
